릴스 캡션 생성기

영상 파일이나 릴스/유튜브 링크를 넣으면 전사문, 화면 프레임, 기존 학습용 캡션 예시를 종합해 인스타그램 게시용 캡션.txt와 이모지 확인용 캡션.html을 만들어 줍니다.

developed by yeohj0710

빠른 사용법

  1. 릴스 캡션 생성기.exe를 실행합니다.
  2. OpenAI API 키를 입력합니다. 키는 입력 중 바로 확인할 수 있고, 기본적으로 이 PC에 저장됩니다.
  3. PC 파일 선택 또는 링크로 가져오기를 선택합니다.
  4. PC 파일을 고른 뒤 옆의 열기로 원본 폴더를 바로 열 수 있습니다.
  5. 저장 위치를 확인하고 캡션 생성 시작을 누릅니다.
  6. 완료되면 캡션 HTML 열기 또는 결과 폴더 열기로 결과를 확인합니다.

결과 저장 위치

PC 파일을 선택하면 기본적으로 영상이 있는 폴더에 캡션.txt와 캡션.html을 저장합니다. 기존 학습용 데이터 폴더 안의 영상을 선택하면 그 폴더에 바로 캡션이 만들어집니다.

캡션.html에는 이모지 표시용 폰트와 캡션 복사 버튼이 들어 있어 TXT 뷰어에서 이모지가 안 보일 때 확인하기 좋습니다.

링크로 가져오면 생성된 캡션 폴더 아래에 새 폴더를 만들고, 영상·스크립트·스크린샷·캡션을 함께 저장합니다.

자동 분석

학습용 데이터 폴더 안의 기존 캡션.txt를 자동으로 참고해 말투, 줄바꿈, 해시태그, 주의 문구 스타일을 맞춥니다.

화면 분석 프레임 간격은 영상 길이에 맞춰 자동으로 정해집니다. 기본적으로 1초 간격에 가깝게 보되, 긴 영상은 최대 60장 안에서 자동으로 간격을 넓힙니다. 화면에 논문명, DOI, 학술지명이 보이면 캡션 하단에 참고 논문을 짧게 포함하도록 설정되어 있습니다.

좋은 캡션을 학습용 데이터/영상 주제/캡션.txt 형태로 계속 넣어두면 이후 결과도 그 톤을 더 잘 따라갑니다.

API 비용

OpenAI API 사용량에 따라 비용이 발생합니다. 진행 로그에는 전사와 캡션 생성의 예상 API 비용이 원화 기준으로 표시됩니다. 표시 금액은 1달러=1,500원 기준의 대략값입니다.

예상 캡션 비용이 높으면 앱이 분석 프레임 수를 자동으로 줄여 비용을 먼저 낮춥니다. 영상 길이, 말의 양, 화면 프레임 수, 선택한 모델에 따라 실제 비용은 달라질 수 있습니다.

지원 입력

  • YouTube 영상/Shorts 링크
  • Instagram 릴스/게시물 링크
  • MP4, MOV, MKV, AVI, WEBM 등 주요 영상 파일
  • MP3, M4A, WAV, FLAC 등 주요 오디오 파일

문제 해결

  • Instagram 링크 다운로드가 실패하면 브라우저에서 로그인한 뒤 브라우저 쿠키 사용을 켜고 다시 시도하세요.
  • OpenAI 오류가 나면 API 키, 결제 상태, 모델명을 확인하세요.
  • 캡션이 화면 내용과 다르면 생성된 스크린샷 추출본과 캡션.html을 같이 확인한 뒤 필요한 문장만 직접 다듬으세요.
