# 릴스 캡션 생성기

PC 영상 파일이나 YouTube/Instagram 릴스 링크를 넣으면 `스크립트.txt`, `스크린샷 추출본`, 인스타그램 게시용 `캡션.txt`와 이모지 확인용 `캡션.html`을 만들어 주는 Windows 프로그램입니다.

기존 `학습용 데이터` 폴더의 캡션 예시를 참고해 말투, 줄바꿈, 해시태그, 주의 문구 스타일을 맞추고, 화면 속 제품명·논문명·DOI 같은 정보는 스크린샷 프레임을 우선해서 읽습니다.

**developed by yeohj0710**

## 다운로드

1. 이 페이지 오른쪽 위의 초록색 `Code` 버튼을 누릅니다.
2. `Download ZIP`을 누릅니다.
3. 내려받은 ZIP 파일의 압축을 풉니다.
4. 폴더 안의 `릴스 캡션 생성기.exe`를 실행합니다.

![GitHub Download ZIP 위치](프로그램%20구성%20파일/github-download-zip.png)

## 사용 순서

1. `릴스 캡션 생성기.exe`를 실행합니다.
2. OpenAI API 키를 입력합니다. 입력값은 바로 보이고, 기본적으로 이 PC에 저장됩니다.
3. `PC 파일 선택` 또는 `링크로 가져오기`를 선택합니다.
4. PC 파일을 고른 경우 옆의 `열기`로 원본 폴더를 바로 확인할 수 있습니다.
5. 저장 위치를 확인합니다. 링크 작업은 기본적으로 `생성된 캡션` 폴더에 저장됩니다.
6. `캡션 생성 시작`을 누릅니다.
7. 완료되면 `캡션 HTML 열기` 또는 `결과 폴더 열기`로 결과를 확인합니다.

학습용 데이터, 화면 분석 프레임 간격, 논문 출처 포함 여부, 계정/인물 기본 메모는 앱이 자동으로 처리합니다. 스크린샷은 기본적으로 1초 간격에 가깝게 보되, 긴 영상은 최대 60장 안에서 자동으로 간격을 넓힙니다. 결과를 검토한 뒤 이상한 문장만 사람이 다듬으면 됩니다.

## 결과 파일

PC 파일은 기본적으로 원본 영상 폴더에 `캡션.txt`와 `캡션.html`을 저장합니다. 이모지가 TXT 뷰어에서 깨져 보이면 `캡션.html`을 열고 `캡션 복사` 버튼으로 본문만 복사하면 됩니다.

```text
영상 폴더/
  영상파일.mp4
  스크립트.txt
  캡션.txt
  캡션.html
  스크린샷 추출본/
    0001_00-00-00.jpg
    0002_00-00-05.jpg
```

링크로 가져온 영상은 `생성된 캡션` 폴더 아래에 새 폴더를 만들고, 영상·스크립트·스크린샷·캡션을 함께 저장합니다.

```text
생성된 캡션/
  2605211710 영상제목/
    영상제목.mp4
    스크립트.txt
    캡션.txt
    캡션.html
    스크린샷 추출본/
```

## 비용

OpenAI API 사용량에 따라 비용이 발생합니다.

앱은 작업 중 로그에 전사/캡션 예상 API 비용을 원화 기준으로 표시합니다. 표시 금액은 1달러=1,500원 기준의 대략값입니다. 예상 캡션 비용이 높으면 화면 분석 프레임 수를 자동으로 줄여 비용을 낮춥니다.

기본 모델은 비용이 낮은 `gpt-5-mini`와 `gpt-4o-mini-transcribe`입니다. 실제 비용은 영상 길이, 말의 양, 화면 프레임 수, 선택한 모델에 따라 달라질 수 있습니다.

## 학습용 데이터

공개 저장소에는 바로 실행해 볼 수 있도록 작은 `캡션.txt`/`스크립트.txt` 예시만 포함합니다.

용량이 큰 학습용 영상 파일과 스크린샷 이미지는 GitHub에 올리지 않습니다. 로컬에서 좋은 캡션이 쌓이면 `학습용 데이터/영상 주제/캡션.txt` 형태로 추가해 두면 다음 생성부터 그 톤을 참고합니다.

## 지원 입력

- YouTube 영상/Shorts 링크
- Instagram 릴스/게시물 링크
- MP4, MOV, MKV, AVI, WEBM 등 주요 영상 파일
- MP3, M4A, WAV, FLAC 등 주요 오디오 파일

Instagram 로그인이 필요한 릴스는 `브라우저 쿠키 사용`을 켠 상태에서 Chrome/Edge 등에 로그인되어 있어야 할 수 있습니다.

## 폴더 구성

| 항목 | 설명 |
| --- | --- |
| `릴스 캡션 생성기.exe` | 실행 파일입니다. |
| `사용설명서.html` | 압축을 푼 뒤 볼 수 있는 자세한 사용설명서입니다. |
| `학습용 데이터` | 캡션 톤을 맞추기 위한 예시 폴더입니다. |
| `생성된 캡션` | 링크 작업 결과가 저장되는 폴더입니다. |
| `프로그램 구성 파일` | 실행에 필요한 파일과 개발 파일입니다. 삭제하거나 이름을 바꾸면 실행이 안 될 수 있습니다. |

## 개발

개발 파일은 `프로그램 구성 파일/개발 파일`에 있습니다.

```powershell
cd "G:\내 드라이브\영상 편집\[공통] 유용한 소스\캡션 생성기\프로그램 구성 파일\개발 파일"
.\run_app.ps1
```

빌드:

```powershell
.\build.ps1
```

빌드가 끝나면 루트 폴더에 `릴스 캡션 생성기.exe`가 생성됩니다.
